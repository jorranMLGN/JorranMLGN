Discord Tanki Bot

This code is a JavaScript Discord bot that listens for the message "Tanki" and responds by calling an async function that makes a request to an API and returns the cheapest 3 gas stations. It then prompts the user for how many liters they want to purchase and calculates the total cost. It also adds a "✅" reaction to the cheapest station. It uses "discord.js" and "axios" libraries.

Mastermind

This code is a game where the user has 10 tries to guess a randomly generated code of 4 colors represented by numbers. The user receives feedback on their guess with "Black" or "White". The game will continue to loop until the user correctly guesses the code or runs out of tries, at which point they will be prompted to play again or end the game.
